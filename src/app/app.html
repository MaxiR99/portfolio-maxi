CafeManager Backend

API REST desarrollada para la gestión integral de una cafetería. El sistema permite administrar usuarios, productos, ingredientes, inventario, compras, ventas y métricas del negocio. El backend fue construido con Java y Spring Boot aplicando una arquitectura en capas, separación de responsabilidades y buenas prácticas de desarrollo.

  • Autenticación y autorización mediante JWT.
  • Gestión de usuarios con roles y permisos: ADMIN, CAJERO, BARISTA, COMPRAS.
  • CRUD de usuarios.
  • Control de acceso mediante Spring Security.
  • Persistencia de datos con Spring Data JPA e Hibernate.
  • Documentación interactiva de la API con Swagger/OpenAPI.
  • Manejo de excepciones personalizadas.
  • DTOs y mappers para separación entre capas.

Diseñar un backend escalable y mantenible aplicando conceptos utilizados en entornos profesionales, como autenticación segura, control de permisos, arquitectura organizada y documentación de APIs.
